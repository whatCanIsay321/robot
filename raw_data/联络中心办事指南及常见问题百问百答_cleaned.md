# 一、 产品类
# 1.1 组件类
# 1.1.1 组件板上为什么有色差？
答：您好，组件在生成过程中的生产工艺上会衍生出许多色系的电池片，属于行业内正常现象。天合光能内部标准把多种电池片划分为 4种色系，从而造成同类色系组件间存在轻微视觉差异；后续产品如有任何质量问题，天合光能将提供10年工艺质保以及 25（30）年线性功率质保。
# 1.1.2 组件板上有白色的斑点是什么情况？
答:您好，组件板上的白点是组件生产过程中印刷网版上自带的激光定位点，目前组件的每个大片电池片都有固定约 4个，不影响组件质量。
# 1.1.3 组件板的等级是怎么区分？
答：您好，您若是购买的是天合原装系统，您的产品配置的组件都是 Q1的一等品。
# 1.1.4 组件背光度怎么不一样？
答：您好，组件背面色差及漏光的组件不影响正常使用，DEC15MC20（II）组件享受天合 30年有限店里输出质保服务，请放心使用。
# 1.1.5 光伏组件要定期清洗吗？
答：您好，由于各地区天气情况及污染程度不同，对组件清洗周期暂不做统一规定。根据脏污程度，灰尘过指测试数据，清洗费，清洗后所提升的发电效率以及效益等测算确定清洗频率。
# 1.1.6 你们的产品配置，组件功率是多少的？
答：您好，我们是整套系统进行销售安装的，公司会根据地区和市场情况 按需配置的。

# 1.3.2 现在光伏电站质保多长时间？
答：您好，目前新安装的全款电站整体质保都是 5年，包含逆变器、并网箱、流量监控、电站保险、运维；其中：组件的 10年工艺外观质保；25年线性功率质保（双波 30年）；支架质保 10年。（20年 4月份之前安装的整体质保 3年）
# 1.3.3 光伏电站组件功率&逆变器配置问题
答：您好，公司会根据客户电站情况对产品配置在超配 20%以内:逆变器行标：NB/T 32004里提到的，本身最大输入功率已经是 1.1倍以上了，一般标称的最大输入功率已经是 1.1倍以上，再乘以 1.1，应该至少 1.2倍；比如 5KW机器，标称最大输入功率一般在 5.5KW以上,意思是不要超过 5.5\*1.1=6.05KW；另外，逆变器还会做相应的保护措施；10.23 国家能源局颁布的最新公告，电站配比：《规范》推荐容配比最高达 1.8：1.0。
2020年11，国家能源局下发2020年5号公告，批准了502项新的业标准，包含关于光伏业的《光伏发电系统效能规范（NB/T10394-2020）》。该标准为推荐标准，2020年1023批准，当开始实施。
这是我国个正式下发的、全放开容配的规范。详见《官宣：容配放最1.8，额定功率按交流侧计》
# 1.3.4 为什么夏天光照好，发电量没有春秋高？
答：您好，我们一般说的组件功率是在温度 25度，光照 1000W/m2条件下的输出功率，如果温度和光照变了，功率都会相应变化，不是一个恒定的值。光照跟功率成正比，光照高功率高，光照低功率低。温度跟功率成反比，温度高功率低，温度低功率高。夏天温度高，组件表面可以达到60-70度，相应的功率会降低 14%-18%。在春秋如果温度在10度，功率会升高 6%。
# 1.3.5 电站的发电数据，逆变器高于电表，出现的差异的原因，正常的差距范围是多少？
答：您好，首先您确认下当地供电公司允许并网前逆变器是否有试运行过，其次存在正常的线损和计量误差，约在 4%左右。线路损耗（逆变器至光伏表间线路损耗）+表计误差（逆变器与光伏表两者的计量误差）。

# 二、 业务模式和合作类型
# 2.1合约购业务
# 2.1.1 什么是合约购，是否是贷款？
答：您好，合约购不是贷款，是金融机构根据用户需求购买光伏电站、并出租给用户的一种业务模式。融资租赁协议期内，用户需按期向金融租赁公司支付租金（用户委托公司代其收取电站收益，并代为支付租金，无需用户操心）；合约期结束后，光伏电站所有权归用户。
# 2.1.2 申请了合约购，为什么要上征信？
答：a.融资租赁是目前国际上最普遍、最基本的非银行金融形式，其合规性受国家金融监管部门监管。因此，具备与中
国人民银行系统对接资格和条件的金融租赁公司开展的融资租赁业务，才能登记至中国人民银行征信系统（业务类型为其他，
不是贷款），以保障交易双方的权益。b.根据央行 2021 年 5 月最新管理要求，个人学历、电信、自来水、税收、民事裁决、行政处分、低保救助等信息均将纳
入新版征信报告，全面健全信用体系；c.按期履约行为将记录在案，有助于提高个人信用。
# 2.1.3 申请合约购电站，为什么要拍视频？
答：拍摄视频是为了确保交易真实、有效，确保用户知晓交易信息，且自主自愿申请办理。
# 2.1.4 申请合约购电站，会不会影响用户申请其它贷款？
答：您好，a.个人良好信用记录有助于提高个人信用额度；b.各人信用额度不一，各地金融机构标准不一，如若用户信用额度较低的话，在部分地区申请信用贷款时可能会受到影响；c.当用户申请贷款受到影响时，及时与渠道商务支持部对接，由公司合作金融机构出面协调，介绍合约购业务性质及还款情况，现已有多例通过沟通协调后成功放款的案例。

# 2.1.7 什么叫融资租赁？
答：您好，融资租赁是目前国际上最普遍、最基本的非银行金融形式。它是指出租人（金融租赁公司）根据承租人(用户)的请求，与第三方(供货商) 订立供货合同，根据此合同，出租人出资向供货商购买承租人选定的设备。同时，出租人与承租人签订《融资租赁合同》，将设备出租给承租人使用，并向承租人按期收取租金。a.租赁期内，设备的所有权归出租人所有，承租人只有使用权；承租人需按期向出租人支付租金；租赁期满后，承租人向出租人支付约定金额的名义货价后，设备所有权转让给承租人。b.作为一种金融形式，融资租赁业务的合规性受国家金融监管部门监管。因此，具备与中国人民银行系统对接资格和条件的金融租赁公司开展的融资租赁业务，将登记至中国人民银行征信系统。

解决办法：按照绑卡失败提示的内容，去银行柜台处理该卡的异常；或者换分享收益卡。若该用户名下任何卡都不可用，换用户绑卡支付。

解决办法：需要用户去中信银行柜面申诉。再不行的，换用户绑卡支付。以上两者均不认可，可反馈财务切换成招行支

（1）额定输出≤30kVA，300mA；（2）额定输出＞30kVA，10mA/kVA；2.如并网点接在漏电保护开关用户侧，建议把并网点更改接在漏保电网侧；
# 5.2.4 断路器频繁跳闸，如何处理？
答：1. 国产断路器应选用额定电流＞1.5In的，预留一定余量；
2. 按公司电气配置要求使用线缆，线径原则上只能大不能小，具体使用线径应根据实际敷设距离来确定；
3. 安装时接线确保紧固牢固；
4. 排查是否逆变器短路导致跳闸，更换故障逆变器；
# 5.2.6 元器件烧毁，什么原因？
答：您好，若是线缆虚接发热容易烧毁，或是铝线未用铜铝转换接头，容易烧毁。
# 5.2.7 逆变器噪音太大，影响正常生活，如何处理？
答：您好，逆变器工作会带动风扇转动为了散热，所以会有声音，出厂都经过检测，符合国家标准。（安装环境正常情况下，若是安装在卧室窗下或是不通风处可以挪机）
# 5.2.8 逆变器风扇时转时不转，是不是坏了？
答：您好，逆变器散热采用智能风冷，逆变器温度高需要散热风扇才会运转，若是有故障逆变器会有预警。
# 5.2.9 逆变器风扇故障，过温降载，如何处理？
答：您好，
①风扇故障有预警：确认是否有异物（是否有蚊虫或者脏东西），如有异物需要清洗干净；如无异物还是故障可以让技术对接确认是否是风扇故障--确认故障进行更换风扇或者根据故障部位进行更换
② 风扇的主要作用是散热
③ 安装的位置知否是通风还是较为封闭的空间，也会影响散热
来电身份是用户：可以看下风扇内是否有异物，有----看下是否可以清理，请确保安全的情况下操作。（建议逆变器直流开关关闭后清理）；如果用户不会或者没有异物可以登记信息反馈运维伙伴上门。
来电身份运维伙伴：电站频繁风扇故障，请确认内部是否有异物卡住，请及时清理；若无，反馈对应 T\* 技术/或者服务经理对接；如果技术对接确认设备故障，请及时提交退换货申请。
# 5.2.10 预警绝缘阻抗低故障，如何处理？
答：a.如果多路组串接入，采用逐一排除法，逆变器直流侧只保留一串组件，开机查看是否继续报错，如果不报错，说
明该串组件绝缘性能良好，如继续报错，说明很有可能是该组串绝缘不符合要求，需要继续排查。b.检查组件，接线盒，直流线缆，逆变器，交流线缆，接线端子等对地有无短路或者绝缘层破坏，接线端子外壳有无松
动，有无水进入。无穿管露天敷设和墙角处的线缆尤其要仔细检查。c.检查系统有没有良好接地。d.如以上排查都无明显问题，需要联系售后。
# 5.2.11 预警继电器故障、相线接地故障、地线故障，如何处理？
答：a.用万用表电压档测量零线对地电压，正常情况下，零线无明显电压，如检测到有电压，排查是否电网本身问题（断
开并网点测量）。b.如电网侧零线电压较低，建议把零线与地线连接，同时确保接地良好。c.如电网侧零线电压在 220V左右，确认火线对地电压，如发现有火线对地电压明显偏低，需要联系当地电网排查该火线
漏电情况。（非专业电工不要触碰任何金属物体，包括组件支架，逆变器、并网箱外壳，接地扁铁、接地极）d.如电网侧零线无电压，需逐一检查并网箱到逆变器交流线路。e.如果排查无明显异常，需要联系售后，禁止对零线检测关闭。
# 注意：正常情况零线对地电压是 0，可能存在±5v 的波动属于正常现象；人体安全电压 36v 以内。
专业解释：在理想的三相四线制供电系统中，零线（中性线）与地线在变压器中性点处是直接接地的，因此两者的电位理论上相等，电压差应接近于零。实际运行中，由于线路阻抗、负载不平衡等因素，可能会存在微小电压差，但一般不超过±5V。
# 5.2.12 交直流端子烧毁是什么原因？
答：您好，安装虚接，或是端子未压牢都容易导致烧毁。
# 5.2.13 TD 机型逆变器显示重连倒计时、并网倒计时，如何处理？
答：您好，后台查看下（确认是否最高安规，若不是，透传最高安规），这边已处理，请确认逆变器是否已经启动；若机器还是不启动，请排查是否零线带电或交流端子是否虚接。
# 5.2.14同村安装，为何我家就老是故障，预警市电电压超范围，我邻居电站就不预警？
答：逆变器至电网并网点以及并网点到变压器之间的线缆过细、过长、或者线缆材质不合格，导致逆变器 AC 端子侧的电压抬升，超过逆变器安规设定的并网电压范围，逆变器显示 “市电电压超范围”报警；
排查参考：1.选择合适规格，并适当增大线径的交流电缆并网，减少线损；
2.选择就近点并网，缩短逆变器到并网点的距离，减少线损；
3.如果并网电缆选择铝线时，对应铝导线的截面积要求至少高于铜导线截面积的 1-2个档位；
# 5.2.15 逆变器屏幕无显示，如何排查？
答：用万用表电压档测量组串电压，红表笔接正极，黑表笔接负极，如果万用表显示电压为正且电压值符合理论值判断正常。如果显示电压为负则判断极性接反。如果显示电压为零，依次检查直流开关，QC4端子，组件连接端子是否正常。如果逆变器是使用一段时间后且以上情况排查过未发现有异常，则可能是逆变器直流侧硬件电路发生故障，需要联系售后。
# 5.2.16 逆变器显示电网过压或欠压，如何排查？
答：用万用表电压档测量电网侧电压，如果超出逆变器设置的电压保护范围，则适当调高电压标准，如果未超出电压保护范围，参考以下步骤。a.检查交流输出端子接线情况，重点关注零线，虚接会影响电压的采样。b.另外由于电网电压过高原因导致的此类问题，如果修改了电压范围还是未能解决，需要考虑加大输出电缆线径或者尽可能把逆变器安装靠近并网点，必要时协调电网适当降低变压器输出电压。
# 5.2.17 逆变器预警漏电流故障，如何排查？
答：如果在天晴时能恢复，考虑是组件对地电容大导致，如并网箱内安装 30mA漏保，建议更换 100mA。如并网点接在漏保用户侧，建议把并网点更改接在漏保电网侧。
如果漏电过大，需要排查线路存在漏电故障，采用逐一排除法，逆变器直流侧只保留一串组件，开机查看是否继续报错，如果不报错，说明该串组件无漏电，如继续报错，需要考虑是否该组串线路存在漏电可能。
如果直流侧无明显问题，使用绝缘电阻测试仪测试交流对地阻抗，检测交流电缆是否有绝缘问题。
如果不能检测出明显问题，需要考虑机器本身存在漏电流过大问题。把直流侧和交流侧全部断开，让逆变器停电 30分钟以上，如开机能恢复就继续使用，如果不能恢复，需要联系售后。
# 5.2.18 逆变器预警电网扰动，如何处理？
答：现场排查交直流接线端子是否虚接，如无明显异常，联系对应的逆变器型号的技术支持
# 5.2.19 逆变器外壳带电，万用表测量外壳对地有较高电压，如何处理？
答：检查逆变器外壳有无单独接地，如果没有，需要使用接地线跟接地极连接起来。如果有，检查接地线路的连续性。如果第一次并网出现此类问题，需确认火线、零线、地线接线位置是否正确。检查交流线路中有没有电缆破损导致的对地漏电。（非专业电工不要触碰任何金属物体，包括组件支架，逆变器、并网箱外壳，接地扁铁、接地极）
# 5.2.20 逆变器不停重连倒计时，无法并网？
答：如果在天气晴好时出现须排查交流侧接线情况。
检查交流侧各端子连接情况，如没有明显问题，把逆变器输出直接接到电网进线侧看是否并网。确认并网箱到电线杆并网点连接是否虚接。
以上如无明显问题，联系售后。
# 5.3 低效能
# 5.3.1低效能如何排查?
答：您好，请按照以下步骤排查a.组件安装容量是否减少；b.阴影遮挡：周边是否存在树木，建筑物，山体等固定物遮挡；阵列前后排间距是否过小;c.组件板上是否有灰尘堆积；d.安装角度是否不是最佳安装倾角或朝向是否不是正南；e.逆变器接入的每个组串是否正常发电；f.逆变器直流侧开关是否全部打开；g.组件是否被雷击，组串电压是否偏低；h.组串是否失配，单串组件数量是否过少，单路 MPPT 上接入组串是否过多；i.当地电网电压是否过高，频繁停电或三相不平衡，零线带电影响；j.逆变器风扇是否工作异常：不转或异物卡住；k.每日发电量是否异常清零；如存在以上问题，请及时联系技术确认现场情况并提供相关检测数据；若是技术判断设备故障，请按要求提交退换货申；

# 5.7 国网数据异常
5.7.1 国网数据异常是什么情况，需要怎么排查？
答：逆变器实际发电量与上网电量偏差大，核对内容就和国网数据异常基本一样：
1、 查国网近 7 日发电量与监控平台近7 日发电比对；
2、查国网月发电量与监控月发电量比对；

# 常规工作要求（前四条使用所有客服人员）：
1.客服专员每班必须提前 10 分钟到岗；
2.客服专员到达座位后，应检查所用电话、电脑以及其它与工作有关的设施，确认是否可以正常使用；
3.如座席硬件设施不能正常使用，由主管根据情况安排员工调整座位；
4.客服专员在进入操作系统前，应将笔、本等与工作有关的所有资料准备齐全；
5.带好耳机，客服专员需登录系统，准备接听客户的电话；
6.电话必须在响铃 3秒内接通
7.进行日常工作，包括业务咨询、投诉受理以及回访服务等；
8.休息交接班时做好地接
9.下班前总结当日疑难问题建议，待处理问题及当班情况，并提交汇总给主管；
10.下班，退出系统并关闭计算机。
接通后：您好，请问有什么可以帮您？
让客户长时间等待后：开始：请您稍等，这边帮您查询下……结束：您好，感谢您的耐心等待！挂机前：请问还有其他可以帮您的吗？/请问，您还有其他需要补充的吗？若无，感谢您的来电，祝您生活愉快，再见！注：不得与客户发生语言冲突，以解决问题为目的，正确引导客户，注意服务态度。
# 7.2.2 接通无声如何规范挂机？
答：正常询问：请问有什么可以帮您的？抱歉听不到您的声音，请问有什么可以帮您？按照话术询问三次后；非常抱歉这边听不到您的声音，如果有需要可以再次来电，感谢来电，再见。挂机后主动回电客户是否有什么需要咨询的，如果仍无声音按照话术，做放弃处理。
# 7.2.3 接通后客户突然异常挂断如何处理？
答：需要给客户回电继续咨询 若呼出三次（每次间隔 5-10 分钟）未接听正常完结工单。
# 7.2.4 客户来电反馈问题，但是不愿意提供电站信息如何处理？
答：非常抱歉！需要您提供正确的电站信息，才能正确快速的核实问题处理问题，如果您暂时无法提供，建议您核实之后再次来电，感谢您的配合，谢谢！
# 7.2.5 客户多次来电反馈同一事情未有人对接处理，如何处理？
答：按照投诉升级的规则进行升级。
# 7.2.6 渠道伙伴来电反馈服务经理不接电话，不回信息如何处理？
答：① 确认客户拨打的电话是否正确，② 对应的区域负责人是谁，是否正确③了解客户诉求，解决客户问题
如果确实没有接听，致歉，说明情况 服务经理经常在外：可能在工地上或者培训现场或者在路上，可能没有听到电话，建议您换个时间再联系，或者我们帮您催促下让服务经理给您回电，给您带来不便非常抱歉。

# 7.4工单审核和派单
# 7.4.1 工单审核和派单要求有哪些？
答：当天推送的工单当天必须审核完成，符合派单规则工单推送：选择接收时间为：前一天至当天的工单推送；推送状态：对推送失败和未推送的工单需要进行人工推送，人工推送后仍然推送失败需要排查具体原因，确认该故障已经处理
