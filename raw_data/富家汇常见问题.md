# 1.1 渠道招募合伙人信息反馈

# 1.1.1 富家汇的信息推广和适用业务范围是什么？

答：这个平台主要是招募合伙人，新合伙人为各品牌的二级分销以及行业外的小白。

# A、登录方式：

$\textcircled{1}$ 天富通 APP--工作台--项目开发--富家汇$\textcircled{2}$ 微信小程序--点击登录注册--勾选用户协议和隐私政策 ---点击账号密码登录 --授权登录

# B、合伙人绑定渠道伙伴的三种方式

$\textcircled{1}$ 渠道伙伴保存二维码发给合伙人，合伙人微信扫码准入，即成为其下级;  
$\textcircled{2}$ 渠道伙伴分享链接至合伙人微信，合伙人点击链接准入，即成为其下级;  
$\textcircled{3}$ 合伙人直接在小程序 “富家汇”注册，扫描渠道伙伴二维码，即成为其下级。

# C、注意事项：

$\textcircled{1}$ 如果富家汇小程序提示“子账号不能注册直营合伙人，请解绑后重新注册”，按培训手册操作  
$\textcircled{2}$ 主账户可直接通过天富通-工作台-富家汇进入，也可通过富家汇小程序选择账号密码(天富通账号)进入;  
$\textcircled{3}$ 子账号通过天富通进行解绑，进入富家汇小程序注册；  
$\textcircled{4}$ 如通过平台推广进入的公共信息，渠道中心会安排专人跟进处理。  
$\textcircled{5}$ 合伙人是否签约不影响合伙人的招募推广工作，合伙人完成注册即可生成二维码/邀请码，通过二维码/邀请码可以招募下级合伙人D、解绑天富通流程（注销前请自行核实数据、金额相关信息，注销后将无法恢复，所带来的损失需自己承担）

$\textcircled{1}$ 使用该手机号登录 天富通 APP$\textcircled{2}$ 点击我的-企业管理-退出当前企业，该手机号会自动注销;老版天富通点击我的-账号安全-手机号-解绑即可。

# 1.2 合伙人注册注意事项

# 1.2.1 合伙人要求是什么？

答：以下情况不能注册富家汇合伙人$\textcircled{1}$ 老赖、失信等人员 $\textcircled{2}$ 年龄大于 18 岁，小于 75 岁

# 1.2.2 合伙人注册步骤

答： $\textcircled{1}$ 小程序搜索富家汇 $\textcircled{2}$ 点击登录注册 $\textcircled{3}$ 勾选用户协议和隐私政策 $\textcircled{4}$ 点击手机号快速登录 $\textcircled{5}$ 授权登录

# 1.2.3 合伙人完成签约的具体标准

答：完成以下事项： $\textcircled{1}$ 选择结算方式、 $\textcircled{2}$ 选择接单区域范围、 $\textcircled{3}$ 实名认证、 $\textcircled{4}$ 协议签署、 $\textcircled{5}$ 培训考试。（结算方式为对公付款，实名认证为企业实名认证加开票信息）

# 1.3 渠道伙伴相关问题

# 1.3.1 渠道伙伴如何注册为富家汇合伙人

答：非子账户的渠道伙伴可以通过以下两种方式注册富家汇：$\textcircled{1}$ 天富通：点击工作台 项目开发 点击富家汇$\textcircled{2}$ 富家汇：微信小程序搜索富家汇 点击登录注册 勾选用户和隐私政策 点击账户密码登录 授权登录

# 1.3.2 系统提示子账户不能注册直营合伙人

答：解绑天富通账户后，后到微信小程序富家汇注册合伙人

# 1.3.3 渠道伙伴有录单及交付需求

答： 1.渠道伙伴承接公海订单的前提：完成直营平台补充协议签署2.富家汇合伙人完成签约才可提交线索，且营业执照只能被校验一次。渠道伙伴多个分账户实际使用的都是同一个营业执照，如何完成签约中的企业认证和开票信息$\textcircled{1}$ 评估渠道伙伴是否有线索录入的需求，如果无线索录入需求可以不进行签约$\textcircled{2}$ 如果有线索录入需求可以另外找一个未注册过富家汇的手机号码注册为富家汇的合伙人，结算方式选择灵活用

# 1.4 线索录入

# 1.4.1 合伙人线索录入的条件

答：合伙人完成签约

# 1.4.2 接单区域范围和线索的现勘范围不一致，是否可以做线索录入

答：不可以，现勘范围需要和接单区域保持一致，非接单区域的线索合伙人不可录入【11 月份上线】

# 1.5 结算相关问题

# 1.5.1 灵活用工和对公付款的区别

答：灵活用工：一般是指个人合伙人，非企业合伙人对公付款：企业合伙人，有营业执照，需提供发票注：请谨慎选择结算方式，结算方式对应提供不同的签约材料。如选择对公付款，签约需要提供营业执照、开票信息等。

# 1.5.2 业务费用及税点

答：个人合伙人通过平台进行结算，由公司承担相关税费，个税由合伙人承担通过灵工平台代扣代缴 $\textcircled{2}$ 企业主体的业务合伙人，后续费用以服务费形式结算，需开具 $6 \%$ 增值税专用发票 奖励兑付：报装通过后结算奖励标准的 $8 0 \%$ ，剩余20%并网后结算

# 1.5.3 信息不一致，可以结算吗

答：身份证姓名、身份证、手机号码、银行卡开户人一致，结算环节需要人脸识别

# 1.5.4 费用结算以哪个时间点为准享受政策

答：以金融复审通过时间为准（目前），后续以初装费的系统校验时间为准【实际待系统上线后通知为准】

# 1.5.5 其他费用明细及税点

答：依照下发至各分公司的渠道直营政策为准
